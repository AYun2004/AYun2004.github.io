# 本章概要

# （一）微分方程的基本概念

# 1. 微分方程

表示未知函数，未知函数的导数与自变量之间关系的方程叫微分方程.

未知函数是一元函数的最高阶的微分方程叫做常微分方程；未知函数是多元函数的微分方程叫偏微分方程。

# 2. 微分方程的阶

微分方程中所出现的未知函数的最高阶导数的阶数叫做微分方程的阶。

# 3. 微分方程的解

若函数代入微分方程能使该方程成为恒等式，这个函数就叫做该微分方程的解。

# 4. 微分方程的通解

微分方程的解中含有任意常数，且任意常数的个数与方程的阶数相同，这样的解叫做微分方程的通解。

# 5. 微分方程的特解

利用初始条件确定了通解中任意常数的值后所得到的解叫做微分方程的特解。

# 6. 初始条件

用来确定通解中任意常数的条件，如 y∣x=x0=y0,y′∣x=x0=y0′,…,y|_{x=x_{0}}=y_{0},y'|_{x=x_{0}}=y_{0}',…,y∣x=x0​​=y0​,y′∣x=x0​​=y0′​,…, 叫做微分方程的初始条件。

# 7. 初值问题

求微分方程满足初始条件的特解问题，叫初值问题。

# 8. 积分曲线

微分方程的解的图形是一条曲线，叫做微分方程的积分曲线。
