# 第一部分 特征值与特征向量的概念

# 定义 5.1

设 A 是 n 阶方阵，如果存在数λ\lambdaλ 和 n 维非零列向量 x，使关系式 Ax=λxAx=\lambda xAx=λx 成立，那么数 λ\lambdaλ 称为方阵 A 的特征值，非零向量 x 称为 A 的对应于特征值λ\lambdaλ 的特征向量。

（1）特征值问题是对方阵而言的；

# 第二节 相似矩阵与矩阵的对角化

# 第一部分 相似矩阵的概念

# 定义 5.4

设 A，B 都是 n 阶矩阵，若有可逆矩阵 P，使P−1AP=BP^{-1}AP=BP−1AP=B，则称 B 是 A 的相似矩阵，或称矩阵 A 与 B 相似。

对 A 进行运算P−1APP^{-1}APP−1AP 称为对 A 进行相似变换，可逆矩阵 P 称为把 A 变成 B 的相似变换矩阵。

# 第二部分 相似矩阵的性质

# 性质 1

相似矩阵有相同的行列式。

# 性质 2

两个相似矩阵或者都可逆，或者都不可逆。若都可逆，则其逆阵也相似。

# 性质 3

相似矩阵都有相同的特征多项式，从而具有相同的特征值。

# 第三部分 矩阵对角化的条件

# 定理 5.1

n 阶矩阵 A 可对角化的充分必要条件是矩阵由 n 个线性无关的特征向量。

# 推论 1

如果 n 阶矩阵 A 的 n 个特征值互不相等，则 A 与对角阵相似。

# 推论 2

设λ1,λ2,⋅⋅⋅,λm\lambda_{1},\lambda_{2},···,\lambda_{m}λ1​,λ2​,⋅⋅⋅,λm​ 是 n 阶矩阵 A 的 m 个互不相等的特征值，其重数分别为s1,s2,⋅⋅⋅,sms_{1},s_{2},···,s_{m}s1​,s2​,⋅⋅⋅,sm​ 且 s1+s2+⋅⋅⋅+sm=ns_{1}+s_{2}+···+s_{m}=ns1​+s2​+⋅⋅⋅+sm​=n。若对应sis_{i}si​ 重特征值λi\lambda_{i}λi​ 有sis_{i}si​ 个线性无关的特征向量 (i=1,2,・・・,m)，则 A 与对角阵相似。
