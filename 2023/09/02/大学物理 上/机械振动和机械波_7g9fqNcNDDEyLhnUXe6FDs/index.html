# 第一部分 振动的概念

振动：

任何一个物理量在某一量值附近随时间作周期性变化。

机械振动

物体在某一稳定平衡位置附近做往复运动。

实例：

心脏的跳动，钟摆，乐器，地震等。

# 机械波

# 第一节 机械波的形成和传播

# 第一部分 波动

振动状态在空间的传播称为波动或波。

振动 —— 波动的成因，波动 —— 振动的传播；

# 第二部分 机械波

机械振动在连续介质中的传播。

1. 形成条件

1）波源 做机械振动的物体（波源，提供能量）

2）连续介质（气体，液体，固体）

2. 分类

1）纵波 —— 质点的振动方向与波的传播方向一致

2）横波 —— 波传播方向上各点的振动方向与波的传播方向垂直

# 第三部分 简谐波

简谐振动在弹性媒介中传播所形成的机械波。

# 第四部分 描述机械波的特征量

1. 振幅 A

2. 波长λ\lambdaλ 相位差为 2π2\pi2π 的两相邻质点之间的距离（一完整波形的长度）

频率: v=1Tv=\frac{1}{T}v=T1​

# 第四节 惠更斯原理 波的叠加和干涉

# 第一部分 惠更斯原理

波动传播到的各点都可以看作是发射子波的新波源；其后任一时刻，这些子波的包迹就是新的波阵面。

# 第二部分 波的衍射

波的传播过程中遇到障碍物时，能绕过障碍物边缘，继续向前传播的现象称为波的衍射，或称为波的绕射。

# 第三部分 波的叠加原理（独立性原理）

几列波在某区域相遇后，传播情况与未相遇时相同，互不干扰。

在相遇区，任一质点的振动为各列波单独在该点引起的振动的合成。

# 第四部分 波的干涉

两列波：相同的振动方向，周期（T），波速（u）

合振动振幅： A=A12+A22+2A1A2cosΔφA=\sqrt{A_{1}^{2}+A_{2}^{2}+2A_{1}A_{2}cos\Delta \varphi}A=A12​+A22​+2A1​A2​cosΔφ​

波的干涉现象：

如果两列波满足一定条件，在空间相遇时，有些点的振动始终是加强的，有些点的振动始终是减弱的，甚至是完全抵消的。

产生干涉的条件：

1）振动方向相同

2）频率相同

3）相位差恒定

相干波：满足干涉条件两列波。

相干波源：产生相干的波源。

波节 振幅为零

相邻波腹（波节）间距 =λ2=\frac{\lambda}{2}=2λ​

相邻波腹和波节间距 =λ4=\frac{\lambda}{4}=4λ​

2. 相邻两波节间各点振动相位相同，一波节两侧各点振动相位相反。

驻波的能量

各质元处在正负最大位移位置时（ E=EPE=E_{P}E=EP​）波腹处：E 最小；波节处：E 最大；

各质元处在平衡位置时（ E=EkE=E_{k}E=Ek​）波腹处：E 最大；波节处：E 最小；

驻波的动能和势能相互转换，不断地分别集中在波腹，波节附近，不向外传播。

# 半波损失

当波从波疏介质垂直入射到波密介质入射波与反射波在介质分界面处的相位时时相反，即反射波在分界处产生 π\piπ 的相位跃变，相当于出现了半个波长的波程差，称半波损失
