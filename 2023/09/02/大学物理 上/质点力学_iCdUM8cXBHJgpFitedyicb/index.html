# 第一部分 牛顿第一定律

任何物体要保持其静止或匀速直线运动状态，直到外力迫使它改变运动状态为止。

1. 几点说明

1）力是改变物体运动状态的原因；

2）任何物体都具有惯性，此定律也称惯性定律；

2. 惯性系

在一个参考系观察，一个不受力作用或处于平衡状态的物体，将保持静止或匀速直线运动的状态，这个参考系叫惯性系。

注意：

凡是对于惯性系做匀速直线运动的一切参考系都是惯性系。

非惯性系

# 第二部分 牛顿第二定律

物体受到外力作用时，他所获得的加速度的大小和外力的大小成正比，并与物体的质量成反比，加速度的方向与外力方向相同。

F⃗=ma⃗   单位：N=kg⋅m⋅s−2\vec{F}=m\vec{a} \ \ \ 单位：N=kg\cdot m\cdot s^{-2} F=ma   单位：N=kg⋅m⋅s−2

说明：

（1）质量：质量 m 是惯性的量度；

（2）瞬时性：定律中的力和加速度都是瞬时的，同时存在，同时消失；

（3）力的叠加性原理：

F⃗=∑F⃗i=ma⃗\vec{F}=\sum \vec{F}_{i}=m\vec{a}F=∑Fi​=ma

（4）分量形式

# 第三部分 牛顿第三定律

两个物体之间的作用力 F⃗\vec{F}F 和反作用力 F⃗′\vec{F}'F′ 沿同一直线，大小相等，方向相反，分别作用在两个物体上

F⃗=−F⃗′\vec{F}=-\vec{F}' F=−F′

两点说明：

（1）作用力，反作用力，分别作用于二物体，各产生其效果；

（2）作用力和反作用力是性质相同的力。

W=12mv22−12mv12=Ek2−Ek1W=\frac{1}{2}mv_{2}^{2}-\frac{1}{2}mv_{1}^{2}=E_{k2}-E_{k1}W=21​mv22​−21​mv12​=Ek2​−Ek1​

合力对质点所作的功，等于质点动能的增量 —— 质点的动能定理

注意：

功和动能依赖与惯性系的选取，但对不同惯性系动能定理形式相同。

# 第三部分 保守力

功是过程量，一般情况下与质点所走过路程有关，但若某个力的功仅与始末位置有关，与路径无关，这样的力称为保守力。

质点沿任意闭合路径运动一周时，保守力对他所做的功为零（ W=∮lF⃗⋅dr⃗=0W=\oint_{l}^{}\vec{F}\cdot d\vec{r}=0W=∮l​F⋅dr=0）

非保守力：

力所作的功与路径有关（例如摩擦力）

# 第四部分 势能

保守力的功，仅取决于物体是始末态的相对位置，可以引入一个位置函数，用 EpE_{p}Ep​ 表示

W=−(Ep2−Ep2)=−ΔEpW=-(E_{p2}-E_{p2})=-\Delta E_{p} W=−(Ep2​−Ep2​)=−ΔEp​

这个函数具有能量的性质；又是由物体相对位置所决定，故把这种能量称之为势能 (位能)

势能计算

1） W=−(Ep2−Ep1)=−ΔEpW=-(E_{p2}-E_{p1})=-\Delta E_{p}W=−(Ep2​−Ep1​)=−ΔEp​

Ep1E_{p1}Ep1​: 初始位置的势能
Ep2E_{p2}Ep2​: 末位置的势能

2） 令 Ep1=0E_{p1}=0Ep1​=0 零势能点 “0” 一般取保守力做功的起点位置

Ep=−W=−∫abF⃗⋅r⃗=∫baF⃗⋅dr⃗=∫b"0"F⃗⋅dr⃗E_{p}=-W=-\int_{a}^{b}\vec{F}\cdot\vec{r}=\int_{b}^{a}\vec{F}\cdot d\vec{r}=\int_{b}^{"0"}\vec{F}\cdot d\vec{r}Ep​=−W=−∫ab​F⋅r=∫ba​F⋅dr=∫b"0"​F⋅dr

势能零点 势能表达式
重力势能 地面处 Ep=∫z0−mgdz=mgzE_{p}=\int_{z}^{0}-mgdz=mgzEp​=∫z0​−mgdz=mgz
弹力势能 弹簧原长处 Ep=∫x0−kxdx=12kx2E_{p}=\int_{x}^{0}-kxdx=\frac{1}{2}kx^{2}Ep​=∫x0​−kxdx=21​kx2
引力势能 无穷远处 Ep=∫r∞−GMmr2dr=−GMmrE_{p}=\int_{r}^{\infty}-\frac{GMm}{r^{2}}dr=-\frac{GMm}{r}Ep​=∫r∞​−r2GMm​dr=−rGMm​

几点说明：

1）只有保守力场才能引入势能。

2）势能大小是相对量所选取的势能零点有关。

3）势能是彼此以保守力作用的系统所共有的。

4）保守力对于以保守力作用的系统是内力。

# 第五部分 质点系的动能定理

对第 i 个质点，有

Wi外+Wi内=Wi=Eki2−Eki1W_{i外}+W_{i内}=W_{i}=E_{ki2}-E_{ki1} Wi外​+Wi内​=Wi​=Eki2​−Eki1​

对质点系，有

∑iWi外+∑iWi内=∑iEki2−∑iEki1=Ek2−Ek1\sum_{i}W_{i外}+\sum_{i}W_{i内}=\sum_{i}E_{ki2}-\sum_{i}E_{ki1}=E_{k2}-E_{k1} i∑​Wi外​+i∑​Wi内​=i∑​Eki2​−i∑​Eki1​=Ek2​−Ek1​

质点系动能定理

W外+W内=Ek2−Ek1W_{外}+W_{内}=E_{k2}-E_{k1} W外​+W内​=Ek2​−Ek1​

物理意义：

系统动能的增量等于质点系外力和内力作功之和。

说明：

内力做功之和不等于零。

# 第六部分 质点系的功能原理

W外+W内=Ek2−Ek1W_{外}+W_{内}=E_{k2}-E_{k1}W外​+W内​=Ek2​−Ek1​

W内=W内保+W内非W_{内}=W_{内保}+W_{内非}W内​=W内保​+W内非​

W内保=−（Ep2−Ep1）W_{内保}=-（E_{p2}-E_{p1}）W内保​=−（Ep2​−Ep1​）

W外+W内非=(Ek2+Ep2)−(Ek1+Ep1)W_{外}+W_{内非}=(E_{k2}+E_{p2})-(E_{k1}+E_{p1})W外​+W内非​=(Ek2​+Ep2​)−(Ek1​+Ep1​)

机械能 E=Ek+EpE=E_{k}+E_{p}E=Ek​+Ep​

W外+W内非=E2−E1W_{外}+W_{内非}=E_{2}-E_{1} W外​+W内非​=E2​−E1​

系统机械能的增量等于外力的功与内部非保守力的功之和。

# 第七部分 机械能守恒定律

当 W外=0且W内非=0W_{外}=0且W_{内非}=0W外​=0且W内非​=0 时，机械能守恒。

说明：

1）W外=0W_{外}=0W外​=0 系统与外界无机械能的交换

2）W非内=0W_{非内}=0W非内​=0 系统内部无机械能与其他能量形式的转换

# 第八部分 能量转换与守恒定律

在一个孤立的系统内，各种形式的能量可以相互转换，但无论怎样转换，这个系统的总能量将保持不变。
