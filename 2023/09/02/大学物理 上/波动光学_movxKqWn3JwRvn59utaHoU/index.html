# 第三部分 光的干涉

1. 光的干涉现象

二列光在相遇区域内，出现稳定分布的明暗交替的条纹。

2. 条件：

频率相同，振动方向相同，相位差恒定 ——> 相干光

3. 相干光的获得

1）分波阵面法

在波源发出的某一波阵面上，取出两部分面元作为相干光源的方法

2）分振幅法：

利用反射，折射把波面上某处振幅分成两部分，再使他们相遇产生干涉现象。

# 光的衍射

# 第一节 光的衍射 惠更斯 — 菲涅耳原理

# 第一部分 光的衍射现象

1. 光的衍射

光在传播过程中遇到障碍物时，不再遵循直线传播的规律，而会绕到障碍物的阴影区域并形成明暗相间的条纹。

2. 条件

障碍物线度与波长数量级相差不多。

缝很小时，衍射现象明显；

缝较大时，光是沿直线传播的。

# 第二部分 惠更斯 — 菲涅耳原理

惠更斯原理：

光波阵面上各点都可以看作发射子波的波源，以后任意时刻，这些子波的包迹就是新的波阵面。

惠更斯原理解释不了光强重新分布。

菲涅耳原理：

（1）从同一波阵面上各点发出的子波是相干波；

（2）子波不向后传播。

# 第三部分 衍射的分类

1. 菲涅耳衍射

障碍物距光源和接收屏距离为有限远。

2. 夫琅禾费衍射

障碍物距光源和接收屏距离为无限远。

3. 分类：

自然光（非偏振光）：

一般光源发出的光，包含各个方向的光矢量在所有可能得方向上振幅都相等。

线偏振光（完全偏振光，偏振光，平面偏振光）：

光矢量只沿一个固定方向振动。

部分偏振光：

某一方向的光振动比与之垂直方向上的光偏振占优势的光为部分偏振光。

产生线偏振方法：偏振片，折射与反射，双折射。

# 第二节 起偏与检偏 马吕斯定律

# 第一部分 偏振片 起偏与检偏

1. 二向色性：

某些物质能吸收某一方向的光的振动，而只让这个方向垂直的光振动通过，这种性质称为二向色性。

2. 偏振片：

涂有二向色性材料的透明薄片。

3. 起偏：

是自然光变成线偏振光的过程。

4 检偏：

检查入射光偏振性的过程。
