# 什么是数学建模

# 原型与模型

# 原型 archetype（原始的模型）

  1. 人们在现实世界里关心，研究或从事生产，管理的实际对象。
  2. 特指文学艺术作品中塑造人物形象所依据的现实生活中的人。
  3. 指在解决问题时，对于新假设的提出有启发作用的那些事物。

# 模型 model

模型是指为了某个特定目的将原型的某一部分信息简缩，提炼构造的原型替代物，是对所研究的系统、过程、事物或概念的一种表达形式。也可指根据实验、图样放大或缩小而制作的样品，般用于展览或实验或铸造机器零件等用的模子。

# 什么是数学建模

# 数学建模的官方解释：

数学建模是利用数学方法解决实际问题的一种实践。即通过抽象、简化、假设、引进变量等处理过程后，将实际问题用数学方式表达，建立起数学模型，然后运用先进的数学方法及计算机技术进行求解。简而言之，建立数学模型的这个过程就称为数学建模。

# 数学建模的分类

  1. 按所用的数学知识分类：

    初等模型，几何模型，微积分模型，微分方程模型，图论模型，概率统计模型，规划论模型等。

  2. 按所解决的问题的领域分类：

    1. 物理模型：自然科学领域内的问题。
    2. 非物理模型：经济模型，交通模型，人口模型，生态模型，环境模型，医学模型，社会学模型。

  3. 按所建模目的分类：

    描述模型，分析模型，预报模型，优化模型，决策模型，控制模型等。

  4. 按所模型的表现特性分类：

    确定模型，随机模型；静态模型，动态模型，离散模型，连续模型。

# 数学建模的一般步骤

  1. 模型假设

    针对问题特点和建模目的；

    做出合理的，简化的假设；

    在合理和简化之间作出折中

  2. 建立模型

    用数学的语言，符号描述问题；

    发挥想象；

    尽量采用简单的数学工具

# 数学建模的全过程

表述：根据建模目的和信息将实际问题 “翻译” 成数学问题。

求解：选择适当的数学方法求得数学建模的解答。

解释：讲数学语言表述为解答 “翻译” 回实际对象。

验证：用现实对象的信息检验得到的解答。
